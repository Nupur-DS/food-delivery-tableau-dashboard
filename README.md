# Food Delivery Data Analysis Dashboard (Tableau)

## Overview
This project presents an interactive Tableau dashboard analyzing food delivery business performance.

## Key Features
- KPI tracking: Total order value, order count, average order value
- Time-based trend analysis
- Top customers and delivery partners analysis
- Product and category-level insights

## Tools Used
- Tableau

## Key Insights
- Identified peak order days and revenue trends
- Highlighted top-performing food items and restaurants
- Provided actionable insights for improving business performance
